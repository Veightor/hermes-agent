# Chutes Prompt Pack

Source: https://chutes.ai
Purpose: Generate brand-consistent imagery, UI visuals, social graphics, product announcements, hero assets, and editorial graphics that feel native to Chutes without copying protected marks or exact site assets.

## 1. Brand Style DNA

Chutes is a cinematic dark AI infrastructure brand: near-black SaaS surfaces, compressed geometric grotesk headlines, restrained developer-product UI, mint “live compute” accents, and a recurring spectral 3D cube motif that turns serverless GPU compute into a modular holographic object. The mood is premium, technical, calm, scalable, and open-source-aligned. Imagery should feel like compute hardware rendered as luminous glass/metal: black environment, purple-blue haze, cyan/magenta/amber iridescence, soft bloom, precise grid/card overlays, and high-trust infrastructure restraint.

Reusable style paragraph:

Dark-mode-native AI infrastructure aesthetic for Chutes: deep black canvas (#0C0C0C), charcoal cards (#121212), off-white typography (#FAFAFA), muted gray metadata (#9C9C9C), sparse mint compute-signal accents (#8BFFC6), and a premium iridescent 3D cube or modular compute object with spectral cyan, blue, indigo, purple, magenta, and amber reflections. Centered SaaS composition, precise grids, thin borders, rounded pill CTAs, subtle purple-blue atmospheric glow, cinematic lighting, clean neo-grotesk typography, developer-native confidence, no cartoon robots, no generic neural-network clip art.

## 2. Palette Lock

UI palette:
- 72% near-black background: #000000, #0C0C0C, #0D0D0D
- 10% charcoal surfaces: #121212, #171717, #1C1C1C
- 10% off-white text: #FAFAFA, #E0E0E0
- 4% muted gray metadata and borders: #9C9C9C, #4A4A4A, #262626, #343434
- 3% mint signal accent: #8BFFC6
- 1% warm/support accents: #FFAB96, #B4C1EE, #22232A

Spectral render palette:
- Cyan: #22D3EE
- Electric blue: #3080FF
- Indigo: #625FFF
- Purple: #A855F7
- Magenta: #F472B6
- Amber: #FBBF24

Color discipline:
- UI should stay mostly monochrome plus mint.
- Spectral color may be vivid, but it should be contained in the cube/object, light bloom, or atmospheric haze.
- Avoid full-rainbow backgrounds. The rainbow lives on material reflections, not the whole page.

## 3. Typography and Copy Overlay Rules

Typography direction:
- Display: Tomato Grotesk-like compressed geometric grotesk; 600 weight; tight line-height; negative tracking.
- Body/UI: Neue Haas Unica-like neo-grotesk; clean, neutral, legible.
- Code/API details: Commit Mono or JetBrains Mono-like monospace.

If image generation includes text, keep it minimal:
- 2-7 words for hero overlays.
- 3-10 words for announcement graphics.
- Use Title Case or short sentence case.
- Avoid long body copy; add abstract text blocks instead if the model struggles with typography.

Good copy snippets:
- Breakthrough Serverless Compute
- Open-Source Models, Live
- Scale AI Without Idle GPUs
- There’s a Chute for Everything
- Hyperscaling AI Products
- Always On. Ready to Scale.
- Trusted Execution Environments
- AI Compute for Everyone

Avoid fake paragraphs, fake logos, and unreadable UI copy. For exact production text, add it later in design software.

## 9. Model-Specific Notes

Midjourney:
Use concise prompts and add aspect/style parameters. Example:

/imagine prompt: cinematic dark AI infrastructure hero for Chutes, single iridescent holographic compute cube floating on near-black canvas, spectral cyan blue indigo purple magenta amber reflections contained in glass material, sparse mint signal glints, purple-blue atmospheric haze, premium serverless GPU compute mood, large negative space for headline, high-end 3D render, precise and calm --ar 16:9 --style raw --v 6

Flux / SDXL:
Use the full descriptive prompt plus the negative prompt. Keep text out of the image when possible; add typography in post. If using LoRA/style references, anchor the palette with #0C0C0C, #121212, #8BFFC6 and the spectral cube material.

OpenAI / DALL-E:
State “no readable text” unless you specifically need a headline. Ask for “a text-safe blank area” rather than rendered typography. DALL-E is good for clean hero/key visuals, less ideal for accurate dense dashboards.

Ideogram:
Use when text accuracy matters. Keep text short, e.g. “AI Compute for Everyone” or “Always On. Ready to Scale.” Ask for compressed geometric sans type in off-white on black.

Runway / Pika video:
Animate subtle cube rotation, slow spectral shimmer, gentle mint signal pulses, and slow parallax through dark UI cards. Keep movement slow and premium; avoid frenetic particle storms.
